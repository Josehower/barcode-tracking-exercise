# SensevenSpa

## Tecnologies

- Next.js
- PostgreSQL

### Requirements:

- Node.js
- PostgreSQL
- pnpm

## setup guide

- Clone The repo
- setup a new Postgres database
- update values on `.env` file following `.env.exmple` template
- install dependencies with `pnpm install`
- run with `pnpm dev`
